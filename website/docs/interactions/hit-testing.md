# Hit Testing System

Hit testing determines what element is under the cursor.

## Built Components
- **HitTester**
- **HitTypes**
- **NodeHit**
- **PortHit**
- **EdgeHit**

## Responsibilities
- Detect node/port/edge on hover
- Detect target port for edge creation
- Return ordered hit results
